Welcome to djust-theming

A complete design system framework for Django — 24 components, 19 color presets, 11 design systems, 6 layout templates, and 8 page templates.

24 Components

Button, card, badge, alert, input, select, textarea, checkbox, radio, modal, dropdown, tabs, table, pagination, breadcrumb, avatar, toast, progress, skeleton, tooltip, nav, sidebar nav, nav item, nav group

19 Color Presets

Default, Shadcn, Blue, Green, Purple, Orange, Rose, Natural20, Catppuccin, Rose Pine, Tokyo Night, Nord, Synthwave, Cyberpunk, Outrun, Forest, Amber, Slate, Nebula

11 Design Systems

Material, iOS, Fluent, Minimalist, Playful, Corporate, Retro, Elegant, Neo-Brutalist, Organic, Dense

6 Layout Templates

Topbar, Sidebar, Sidebar+Topbar, Centered, Dashboard, Split — extend and fill blocks for instant page structure

8+ Page Templates

Login, Register, Password Reset, Password Confirm, 404, 403, 500, Maintenance, Empty State — drop-in themed page fragments

Django Form Integration

Render Django forms with themed styling. Stacked, horizontal, and inline layouts. Automatic error display with themed alerts.

Theme Packs

Complete bundles combining design system + colors + icons + animations + patterns into cohesive themes

Gallery, Storybook & Editor

Built-in gallery to browse all themes, storybook for component isolation, and live editor to customize themes with diff export

Tailwind + shadcn/ui

Generate Tailwind configs, import/export shadcn themes, use theme colors as Tailwind utilities

Architecture Overview

Design Systems vs. Color Presets

Design systems control structure (typography, spacing, shadows, animations). Color presets control palette. Mix any design system with any color preset.

11 x 19 = 209 unique combinations out of the box.

Quick Start

1. Install

pip install djust-theming

2. Configure

{% verbatim %}INSTALLED_APPS = ['djust_theming', ...] TEMPLATES = [{ 'OPTIONS': { 'context_processors': [ 'djust_theming.context_processors.theme_context', ], }, }] LIVEVIEW_CONFIG = { 'theme': { 'preset': 'blue', 'default_mode': 'system', 'critical_css': True, } }{% endverbatim %}

3. Use in Templates

{% verbatim %}{% load theme_tags theme_components %} {% theme_head %} {% theme_switcher %} {% theme_button "Click me" variant="primary" %} {% theme_alert "Done!" variant="success" %} {% endverbatim %}
