Icon Styling Note

Theme packs define icon style guidelines (outlined, filled, rounded, sharp, thin with specific stroke widths), but SVG icons with inline attributes cannot be dynamically restyled via CSS. For real icon style changes, you would need to:

  • Use an icon library with CSS-controllable variants (like Lucide, Heroicons sets)
  • Generate different icon sets matching each pack's specifications
  • Use Web Components or JavaScript to swap icon SVG content

The theme pack system excels at controlling: colors, typography, spacing, animations, patterns, and interactions - all the styling that CSS custom properties can effectively manage!
