What Makes Themes Different?

Unlike color presets which only change colors, themes change the entire design system

Typography

  • • Font families (system vs. custom)
  • • Font sizes and scales
  • • Line heights
  • • Font weights

Layout & Spacing

  • • Spacing scales (tight/normal/loose)
  • • Border radius (sharp/rounded/pill)
  • • Shadow depths
  • • Container widths

Components

  • • Button styles (solid/outlined/ghost)
  • • Card styles (elevated/outlined/flat)
  • • Input styles (outlined/filled/underlined)
  • • Animation timing
