Usage Guide

Page templates are designed to be used inside layout templates. They render page fragments, not full HTML documents.

Auth Page Example

{% verbatim %}{% extends "djust_theming/layouts/centered.html" %} {% load theme_pages %} {% block centered_content %} {% theme_login_page action="/auth/login/" forgot_password_url="/auth/reset/" register_url="/auth/register/" %} {% endblock %}{% endverbatim %}

Error Page Example

{% verbatim %}{% extends "djust_theming/layouts/centered.html" %} {% load theme_pages %} {% block centered_content %} {% theme_404_page home_url="/" title="Oops! Page not found" description="We looked everywhere." %} {% endblock %}{% endverbatim %}
