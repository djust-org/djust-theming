Getting Started

Install, configure, and start building themed Django applications with djust-theming and djust-components.

1. Install

terminal
$ pip install djust-theming djust-components

2. Configure

settings.py 
{% verbatim %}INSTALLED_APPS = [
    'djust_theming',
    'djust_components',
    ...
]

LIVEVIEW_CONFIG = {
    'theme': {
        'preset': 'djust',
        'default_mode': 'system',
    }
}{% endverbatim %}

3. Use in Templates

template.html 
{% verbatim %}{% load theme_tags djust_components %}
<head>
    {% theme_head %}
</head>
<body>
    {% dj_button label="Save" variant="primary" %}
    {% alert variant="success" %}Done!{% endalert %}
    {% badge label="New" status="success" %}
</body>{% endverbatim %}
