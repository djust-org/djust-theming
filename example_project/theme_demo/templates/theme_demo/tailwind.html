Tailwind CSS Integration

Use theme colors directly in Tailwind classes or with @apply directives

Generate Configuration

Create a tailwind.config.js file that uses your theme colors

python manage.py djust_theme tailwind-config --preset blue --output tailwind.config.js

This generates a Tailwind config that maps CSS variables to Tailwind utilities

Using Theme Colors in Tailwind

All theme colors are available as Tailwind utilities

Background Colors

bg-primary
bg-secondary
bg-accent
bg-muted

Borders

border-primary
border-border
border-input

Text Colors

text-primary - Primary text color

text-secondary - Secondary text color

text-muted-foreground - Muted text

text-destructive - Error text

Opacity Modifiers

bg-primary/100
bg-primary/75
bg-primary/50
bg-primary/25
<button class="bg-primary text-primary-foreground hover:bg-primary/90 px-4 py-2 rounded-md"> Tailwind Button </button> <div class="bg-card text-card-foreground border border-border rounded-lg p-6"> Tailwind Card </div>

Using @apply

Create custom CSS classes using theme colors with @apply

Generate Examples

python manage.py djust_theme generate-examples --output theme-components.css

Example CSS

/* Custom button using @apply */ .my-button { @apply bg-primary text-primary-foreground px-4 py-2 rounded-md; @apply hover:opacity-90 transition-opacity; } /* Custom card using @apply */ .my-card { @apply bg-card text-card-foreground rounded-lg border border-border; @apply shadow-sm p-6; } /* Alert variants */ .alert-success { @apply bg-success/10 text-success border border-success/20 rounded-md p-4; }
