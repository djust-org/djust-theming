Django Form Integration

Render Django forms with themed styling using {% verbatim %}{% load theme_form_tags %}{% endverbatim %}. Supports stacked, horizontal, and inline layouts.

Stacked Layout (Default)

Fields stack vertically with labels above inputs. The default and most common layout.

{% csrf_token %} {% theme_form contact_form layout="stacked" %}
{% theme_button "Submit" variant="primary" %}
{% verbatim %}{% load theme_form_tags %}
{% csrf_token %} {% theme_form form layout="stacked" %}
{% endverbatim %}

Horizontal Layout

Labels and inputs side-by-side, suited for wider screens and shorter forms.

{% csrf_token %} {% theme_form login_form layout="horizontal" %}
{% theme_button "Sign In" variant="primary" %}
{% verbatim %}{% theme_form form layout="horizontal" %}{% endverbatim %}

Inline Layout

Compact inline rendering for search bars and quick filters.

{% csrf_token %} {% theme_form login_form layout="inline" %} {% theme_button "Go" variant="primary" %}
{% verbatim %}{% theme_form form layout="inline" %}{% endverbatim %}

Form Validation Errors

Field errors and non-field errors are displayed with themed alert styling.

{% csrf_token %} {% theme_form error_form layout="stacked" %}
{% theme_button "Submit" variant="primary" %}
{% verbatim %}{% load theme_form_tags %} {# Errors are rendered automatically by theme_form #} {# Or render just errors: %} {% theme_form_errors form %}{% endverbatim %}

CSS Prefix

Use {% verbatim %}{% get_css_prefix as prefix %}{% endverbatim %} to access the configured CSS prefix for custom themed markup.

{% verbatim %}{% load theme_form_tags %} {% get_css_prefix as prefix %}
{% endverbatim %}

Form Tag Reference

{% verbatim %}{% theme_form form %}{% endverbatim %} Render a full form with themed layout
{% verbatim %}{% theme_form_errors form %}{% endverbatim %} Render form-level non-field errors
{% verbatim %}{% get_css_prefix as var %}{% endverbatim %} Expose css_prefix for custom markup
